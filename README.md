# Money Manager 📈✨

A premium, offline-first personal finance tracker for Android built with **Jetpack Compose**, **Room Database**, and **Material 3**.

## Features 🚀
- **Dashboard**: Glassmorphism design showing Total Balance, Income, and Expense.
- **Transactions**: Add income or expense with manual date selection.
- **Search & Filter**: Search transactions by note, amount, or date.
- **Safety**: Confirmation dialog before deleting any transaction.
- **Privacy**: Fully offline – all data stays on your device.
- **Premium UI**: Modern dark theme with smooth animations and glassmorphism.

## Tech Stack 🛠
- **Kotlin**: Primary programming language.
- **Jetpack Compose**: Modern toolkit for building native UI.
- **Room**: Local data persistence (SQLite).
- **Material 3**: Latest Material Design components.
- **MVVM Architecture**: Clean and maintainable code structure.

## How to Build 🏗
1. Open the project in **Android Studio**.
2. Click on **Build** > **Build APK(s)**.
3. Locate the `app-debug.apk` and install it on your Android device.
